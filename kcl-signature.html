Outlook Web (your KCL address):
1. Click ① Select for Outlook Web — the signature is highlighted.
2. Press Ctrl+C to copy.
3. In Outlook Web → Settings → Mail → Compose and reply → Signatures → Edit → click in the editor → Ctrl+V.
Outlook's editor is rich-text only — it won't accept raw HTML code, so paste the rendered content this way.

Other clients (Roundcube etc.): use ② Copy raw HTML and paste into the HTML source editor.
